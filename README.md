### Мавзу 2: JavaScript асослари (2 ҳафта)

#### Ҳафта 1: JavaScript синтаксининг асослари ва бошқарув конструкциялари

##### 1. JavaScript синтаксининг асослари (3 кун)

1. **Ўзгарувчилар ва маълумот турлари**:
   - `var`, `let`, `const`
   - Примитив маълумот турлари: `number`, `string`, `boolean`, `null`, `undefined`, `symbol`
   - Мураккаб маълумот турлари: `object`, `array`, `function`
2. **Операторлар**:
   - Арифметик операторлар: `+`, `-`, `*`, `/`, `%`
   - Таққослаш операторлари: `==`, `===`, `!=`, `!==`, `<`, `>`, `<=`, `>=`
   - Мантиқий операторлар: `&&`, `||`, `!`
   - Тенглик ва таъриф операторлари: `=`, `+=`, `-=`, ва ҳ.к.

##### 2. Бошқарув конструкциялари (3 кун)

1. **Шарт операторлари**:
   - `if`, `else if`, `else`
   - Тернар оператор: `? :`
2. **Цикллар**:
   - `for`
   - `while`
   - `do...while`
   - Массивлар ва объектлар билан `for...in` ва `for...of`

#### Ҳафта 2: JavaScript функциялари ва DOM билан ишлаш

##### 3. Функциялар (3 кун)

1. **Функцияларни эълон қилиш**:
   - Оддий функциялар: `function`
   - Ифодали функциялар: `const myFunction = function() {...}`
   - Ўқ функциялари: `const myArrowFunction = () => {...}`
2. **Функция параметрлари ва қайтариш қиймати**:
   - Параметрларни бериш
   - Қайтариш қиймати: `return`
3. **Замонавий функция хусусиятлари**:
   - Ҳамёнига ҳар қандай сонидаги параметрларни олиш: `...rest`
   - Мажбурий параметрлар ва стандарт қийматлар

##### 4. DOM билан ишлаш асослари (4 кун)

1. **Элементларни танлаш**:
   - `document.getElementById`
   - `document.querySelector` ва `document.querySelectorAll`
2. **Элементларни ўзгартириш**:
   - Мазмунни ўзгартириш: `textContent`, `innerHTML`
   - Стилларни ўзгартириш: `style`
   - Атрибутлар билан ишлаш: `setAttribute`, `getAttribute`
3. **Ҳодисаларни бошқариш**:
   - Ҳодиса тингловчилар: `addEventListener`
   - Оддий ҳодисалар: `click`, `mouseover`, `input`, ва ҳ.к.
   - Ҳодисаларни бошқариш функциялари
